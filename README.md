# ⚔️ Code Duel Arena

Code Duel Arena is a real-time competitive coding platform inspired by LeetCode contests, built for live 1v1 coding battles.

Players compete against each other by solving the same DSA problems in real-time. The platform features matchmaking, rankings, live code execution, multiplayer battle rooms, and a modern developer-focused experience.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/code-duel-arena.git
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## Backend Setup

```bash
cd server
npm install
npm run dev
```
